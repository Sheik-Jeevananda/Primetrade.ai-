## Tech Stack

### Frontend
- React.js
- TypeScript
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication

---

## Features

- User Registration & Login
- JWT Authentication
- Role-Based Access (Admin/User)
- Protected Routes
- CRUD Operations for Tasks
- Error Handling Middleware
- Request Validation
- Swagger API Documentation
- Responsive UI

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

Create `.env`:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```
